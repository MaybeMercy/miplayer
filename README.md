米波音乐播放器
=============

这是一款Android音乐播放器，基于Android的原生的MediaPlayer, 使用Android Studio开发。

要求
-------

- Android SDK 21
- Gradle 2.2.1

开始
---

可以使用 "gradlew build" 命令或者在Android Studio中 "Import Project" 自动构建项目。
